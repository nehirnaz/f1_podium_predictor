# F1 Podium Predictor

Machine learning model predicting top-3 podium finishes in Formula 1 using logistic regression.

## Tech Stack
- Python 3.x
- Pandas
- scikit-learn

## Dataset
Kaggle Comprehensive Formula 1 Dataset (2020-2025)

## Project Status
In Progress

## Features (Planned)
- Binary classification: podium vs non-podium
- Points prediction (top 10 finishes)
- Race-level podium predictions

## Usage
```bash
pip install -r requirements.txt
python src/baseline_podium_model.py
